# Python Ödevi

Bu repo, temel Python konularını içeren birkaç küçük alıştırmayı içeriyor. Operatörler, döngüler ve string işlemleriyle ilgili görevleri yaptım.

## 🔹 Görevler

### 1. İki Sayı ile İşlemler
Kullanıcıdan iki sayı alıp bunlarla:
- Toplama
- Çıkarma
- Çarpma
- Bölme
- Mod alma
- Üs alma  
işlemleri yapılıyor.

### 2. 1’den Girilen Sayıya Kadar Toplama
Girilen bir sayıya kadar olan tüm sayıların toplamı hesaplanıyor.

### 3. 1-100 Arası Çift Sayılar
1 ile 100 arasındaki çift sayılar yazdırılıyor.

### 4. Metni Ters Çevirme
Girilen bir kelime veya cümle döngü kullanarak ters çevriliyor.
